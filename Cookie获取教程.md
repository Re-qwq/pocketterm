# Cookie (sauth_json) 获取教程

## 什么是 sauth_json?

sauth_json 是网易我的世界的登录凭证，相当于你的"Cookie"。PocketTerm 需要它来登录游戏。

## 获取方式

### 方式一：使用 MPay 工具（推荐）

项目 `tools/` 目录下提供了 MPay 工具，包含两个版本：

| 文件夹 | 适用渠道 | 登录方式 |
|--------|---------|---------|
| `MPay_163` | 网易官方 | 163邮箱账号密码 |
| `MPay_4399` | 4399渠道 | 4399账号密码 |

#### 步骤

1. **安装前置环境**
   - 下载并安装 [.NET Framework 4.7.2 运行库](https://dotnet.microsoft.com/download/dotnet-framework)（Windows 自带，一般不需要额外安装）
   - 如果提示缺少 DLL，安装 [微软 Visual C++ 运行库](https://aka.ms/vs/17/release/vc_redist.x64.exe)

2. **选择对应文件夹**
   - 163邮箱登录 → 打开 `MPay_163` 文件夹
   - 4399账号登录 → 打开 `MPay_4399` 文件夹

3. **运行登录工具**
   - 双击 `FormLogin.exe`
   - 弹出登录窗口，输入你的账号密码

4. **处理手机验证（如有）**
   - 登录过程中可能要求发送短信验证
   - 按提示将指定数字发送到指定号码
   - 发送后等待验证完成

5. **获取 sauth_json**
   - 登录成功后，窗口中会显示一长串文本
   - 这就是你的 sauth_json
   - 全选复制

6. **导入 PocketTerm**
   - 在 PocketTerm 网页界面中创建机器人
   - 认证方式选择"sauth_json 认证"
   - 粘贴刚才复制的内容

### 方式二：从游戏启动器获取

如果你已经在电脑上安装了网易我的世界启动器：

1. 打开启动器并登录
2. 启动游戏
3. 在游戏安装目录下查找日志文件
4. 日志中搜索 `sauth_json` 关键字
5. 找到后复制整个 JSON 字符串

### 方式三：使用 Fatalder API（需要 Fatalder 账号）

1. 在 PocketTerm 创建机器人时选择"Fatalder API 认证"
2. 输入你的 Fatalder API Key
3. 系统会自动获取 sauth_json

## 注意事项

### 安全提示
- **sauth_json 是你的账号凭证，不要发给任何人**
- 别人有了 sauth_json 可以直接登录你的账号
- 建议使用小号或游客账号

### 使用限制
- 4399 的 sauth_json 有时效性，过期后需重新获取
- 如果登录时显示"服务器维护"，尝试重新获取 sauth_json
- 如果导入器显示账号已被封禁，说明该账号已被网易封号

### 多账号管理
- 每个账号需要单独获取 sauth_json
- PocketTerm 会为每个账号自动生成独立的设备指纹
- 不同账号不会共用同一设备信息，避免被反作弊检测

### 关于手机验证
- 首次使用 4399 登录可能需要手机验证
- 按提示将验证码发送到指定号码
- 验证通过后后续登录通常不需要再次验证
- 手机验证由网易服务器处理，PocketTerm 不存储手机号

## 常见问题

### Q: 登录后显示"您已被禁止登录游戏"
A: 该账号已被网易封禁。请使用新的账号获取 sauth_json。

### Q: sauth_json 多久过期？
A: 4399 渠道的 sauth_json 通常有效期较短（几小时到几天），163 渠道的有效期较长。过期后需要重新获取。

### Q: 为什么名字以 "Nan" 开头？
A: "Nan" 是 MPay 工具注册账号时自动生成的昵称前缀，与 PocketTerm 无关。这是网易的账号命名规则。

### Q: 机器人登录会挤掉正在玩的人吗？
A: 会。如果该账号有人在用，机器人登录后原来的一方会被挤掉。建议使用专用账号。

### Q: 发短信要钱吗？
A: 正常短信费用，由运营商收取。通常一条短信 0.1 元。
