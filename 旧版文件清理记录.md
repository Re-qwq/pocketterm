# PocketTerm 旧版残留文件清理记录

- 清理日期: 2026-07-21
- 当前前端版本: v2 (`frontend/index.html` + `frontend/js/v2app.js` + `frontend/css/main.css`)

## 一、清理前安全核查

删除前已逐项确认无任何活跃代码引用旧版文件:

1. **`frontend/index.html`**
   - 仅引用 `/css/main.css`、`/js/v2app.js`(均为当前版本)
   - 另引用 Google Fonts、Font Awesome CDN
   - 不引用 `dashboard_old.html`、`style.css`、`app.js`、`main.js`、`pages/*.js`

2. **`frontend/js/v2app.js`**
   - 无 `import` / `require` / 动态脚本加载语句
   - 不引用任何旧版 JS 文件
   - (注: 文件中出现的 `.style.cssText` 为浏览器 DOM API,非对 `style.css` 文件的引用)

3. **`backend/app/main.py`**
   - 根路由 `/` 仅返回 `frontend/index.html`
   - 静态资源按目录挂载 `/css` -> `frontend/css`、`/js` -> `frontend/js`(通用挂载,不针对具体旧文件)
   - 不引用 `dashboard_old.html`、`style.css`、`app.js`、`main.js`、`pages/`

4. **全项目搜索**
   - `dashboard_old`: 无任何引用
   - `style.css`: 仅被 `dashboard_old.html`(本次一并删除)及两份文档 `.md` 文本提及
   - `app.js`: 仅被 `dashboard_old.html`(本次一并删除)及一份文档 `.md` 文本提及
   - `main.js`: 无任何引用
   - `pages/*.js`: 无任何引用

结论: 删除操作对当前 v2 前端与后端运行无影响。
